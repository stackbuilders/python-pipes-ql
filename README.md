# Quito Lambda - Creating "Pipes" in Python

## Requirements
- Python
- [asdf](https://asdf-vm.com/)
- [poetry](https://python-poetry.org/)

## Start

```
pip install poetry
poetry install
```

## Run tests for `pipes` library

```
poetry run python test_pipes.py
```

## Run our own `pipes` code

```
poetry run python my_pipes.py
```
